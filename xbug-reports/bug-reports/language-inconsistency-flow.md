# Bug Report: Language inconsistency in flow

## Title
Questions displayed in Swedish when English is selected

## Environment
- Platform: iOS
- App type: Mobile healthcare application
- Language: English

## Preconditions
- User is logged in
- App language is set to English

## Steps to Reproduce
1. Open the app
2. Set the language to English
3. Enter a care flow
4. Continue through the questions

## Actual Result
The questions are displayed in Swedish, while other interface elements are displayed in English.

## Expected Result
All content in the flow should be displayed in English.

## Severity
Medium
