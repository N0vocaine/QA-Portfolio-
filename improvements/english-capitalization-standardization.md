# Improvement: Standardize English Capitalization (Title Case)

## Description
English labels across the application use inconsistent capitalization.

Examples:
- "Cancel booking/rebook"
- "Male hair Loss"
- "Medical certificate"

This creates inconsistency and reduces UI clarity.

## Suggestion
Apply Title Case consistently across all menu items and flows.

## Expected Result
All labels follow the same capitalization pattern:
- Medical certificate → Medical Certificate
- Cancel booking/rebook → Cancel Booking / Rebook
- Male hair Loss → Male Hair Loss

## Impact
- Improves readability
- Creates a consistent user experience
- Aligns UI with common design standards

## Type
UX / Content consistency improvement
