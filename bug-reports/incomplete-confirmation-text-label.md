# Bug Report: Incomplete confirmation text displayed in option label

## Title
Confirmation text is truncated ("I confi") in option label

## Environment
- Platform: Android, iOS
- Devices: Samsung Galaxy S21 5G, iPhone 13
- App version: 4.0.x (Production)

## Preconditions
- User is logged in
- User has an active subscription (AH+)
- App language is set to English

## Steps to Reproduce
1. Enter the "Foul-smelling discharge / genital fungus" flow
2. Go through the questions
3. Reach the confirmation screen ("Please confirm that you have read this")
4. Observe the option label

## Actual Result
The option label is displayed as “I confi” and is truncated.

## Expected Result
The full confirmation text “I confirm” should be clearly displayed without truncation.

## Impact
- Reduces readability and clarity of the confirmation step
- May cause confusion for users before completing the flow
- Impacts overall UI quality

## Severity
Medium
