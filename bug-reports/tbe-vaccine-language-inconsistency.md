# Bug Report: Language inconsistency in TBE vaccine flow

## Title
Questions displayed in Swedish when English is selected in TBE vaccine flow

## Environment
- Platform: Android, iOS
- Devices: Samsung Galaxy S21 5G, iPhone 13
- App version: 4.0.x (Production)

## Preconditions
- User is logged in
- User has an active subscription (AH+)
- App language is set to English

## Steps to Reproduce
1. Set the app language to English
2. Navigate to the TBE vaccine flow
3. Enter the flow
4. Go through the questions

## Actual Result
Questions are displayed in Swedish, while buttons and notifications are displayed in English.

## Expected Result
All content within the flow should be displayed consistently in the selected language (English), including questions, buttons, and notifications.

## Impact
- Creates confusion for English-speaking users
- Breaks language consistency across the flow
- Reduces usability and trust in the application

## Severity
High
