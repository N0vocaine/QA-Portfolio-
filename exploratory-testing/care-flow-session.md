# Exploratory Testing Session: Care Flow

## Objective
Explore the care flow to identify usability issues, inconsistencies, and edge cases.

## Scope
- Entry into flow
- Question navigation
- Language consistency
- Error handling
- Exit and resume behavior

## Test Approach
Unstructured exploratory testing using a user-focused approach.

## Areas Explored
- Switching language mid-flow
- Navigating back and forward between questions
- Leaving the flow and returning
- Handling incomplete answers
- Accessing ongoing errands

## Findings

### 1. Language inconsistency
Some questions are displayed in Swedish while the app language is set to English.

### 2. Navigation confusion
Entering an ongoing errand shows an extra step before reaching the conversation, creating unnecessary friction.

### 3. Missing validation feedback
Some fields allow continuation without clear validation messages.

## Risks Identified
- Confusion for non-Swedish users
- Increased drop-off in flows
- Reduced trust in the product

## Conclusion
The flow works functionally, but inconsistencies and UX issues impact the overall user experience.
