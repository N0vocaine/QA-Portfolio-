# QA Portfolio

Hello, I am Adriana and this is my QA portfolio.

I am a Software Testing student currently doing an internship at Doktor.se in Stockholm, where I work with testing mobile and web healthcare applications in an Agile environment.

I analyse user stories and acceptance criteria, design and execute test cases, and perform exploratory and regression testing across different user flows. I also report and track defects in Jira and Linear, and collaborate with developers to clarify issues and verify fixes.

I focus on understanding the full user journey, identifying risks, and ensuring that the application behaves correctly across different states.
